# Portafolio de Ingeniería Mecatrónica - Industria 4.0
¡Bienvenido a mi repositorio de desarrollo e investigación! Soy Ingeniero Mecatrónico recién egresado, enfocado en la convergencia de la automatización industrial, el Internet de las Cosas (IIoT) y la analítica de datos.

Este espacio compendia un plan de entrenamiento intensivo diseñado para implementar soluciones de la Industria 4.0 y prepararme para estudios avanzados de Maestría (R&D).

## 📂 Estructura del Repositorio

* **`01-Edge-IoT-Gateway/`**: Desarrollo de scripts en Python, contenedores Docker, flujos de Node-RED y protocolos de comunicación (MQTT, Modbus TCP).
* **`02-Control-PLC-SCADA/`**: Lógica de programación de PLC en Texto Estructurado (IEC 61131-3) usando Codesys e interfaces HMI/SCADA con Ignition.
* **`03-Digital-Twin-Analytica/`**: Modelado CAD cloud-native, simulación física y algoritmos de mantenimiento predictivo.
* **`00-Documentacion-Academica/`**: Notas de investigación, resúmenes de papers científicos y borradores en formato IEEE.

## 🛠️ Tecnologías y Herramientas
* **Lenguajes:** Python, C++, Texto Estructurado (ST).
* **Infraestructura & IoT:** Docker, Node-RED, Mosquitto MQTT, ESP32.
* **Automatización:** Codesys, Ignition SCADA, OPC UA.
* **Productividad:** VS Code, Git/GitHub.
